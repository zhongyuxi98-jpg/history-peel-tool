# Student Manual v1.0

## Workflow

1. **Initialize**
   Use Prompt 1 to choose a question.

2. **Build**
   - [Blue] Point Sticker (Meta)
   - [Colored] Content & Evidence
   - [Purple] Mechanism Links
   - [Gray] Link Back Sticker

3. **Audit**
   Copy text to AI for Prompt 2 (structure check).

4. **Draft**
   Use Prompt 3 to generate the 120-word essay paragraph.
